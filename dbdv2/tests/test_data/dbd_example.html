หมายเหตุ

  1. ข้อมูลนิติบุคคล : ข้อมูลนี้เป็นเพียงข้อมูลทั่วไปที่เปิดเผยเท่านั้น เพื่อประโยชน์ในการให้บริการประชาชน ไม่สามารถนำไปใช้อ้างอิงได้ตามกฎหมาย 
  2. ข้อมูลงบการเงิน : ข้อมูลนี้เป็นข้อมูลที่ได้จากการนำส่งงบการเงินของนิติบุคคล ข้อเท็จจริงเป็นสิ่งที่ควรหาไว้พิจารณาฐานะ
  3. กรณีที่ยังไม่เคยมีการจัดส่งงบการเงิน ระบบจะนำเสนอวัตถุประสงค์ตอนจัดตั้งกิจการ หรือที่ปรากฏในหน้าหนังสือรับรองฯ
  4. ประเภทธุรกิจที่ส่งงบการเงินปีล่าสุด หมายถึง ประเภทธุรกิจของกิจการตามที่ระบุในงบการเงินปีล่าสุดที่จัดส่ง
  5. ค่ามัธยฐานของประเภทธุรกิจคำนวณจากจำนวนนิติบุคคลในประเภทธุรกิจ 62022 ที่ส่งงบการเงินในปี 2562 และมีขนาดเล็ก (S)* ซึ่งมีจำนวนทั้งสิ้น 178 ราย
    *ดูหลักเกณฑ์การจัดกลุ่มขนาดธุรกิจ
